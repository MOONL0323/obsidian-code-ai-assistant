# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-06

### Added
- Initial release
- OpenAI integration (GPT-4, GPT-4o, GPT-4o-mini, GPT-3.5-turbo)
- Anthropic integration (Claude 3.5 Sonnet, Claude 3 Opus, Claude 3 Haiku)
- Streaming response support
- Token counting and cost estimation
- Local Git repository management using simple-git
- GitLab API integration
- File tree browser with depth limit and filtering
- Branch management and switching
- Multi-language code file reading
- Chat view with message history
- Settings panel for configuration
- Chinese and English language support
- Logging system

### Technical Details
- Built with TypeScript 5.3.0
- Bundled with esbuild 0.19.0
- Based on Costrict architecture
- Clean and modular codebase
